Credit Card Fraud Detection with Machine Learning

Timothy LaRoy Jr.

For my B.S. capstone project I chose to develop a tool that allowed a fictional bank employee to input a file of transaction data and receive a report of suspected fraudulent transactions. The report was generated ontop of a random-forest model that used the banks historical transactional data to train. The model acheived over 99% accuracy. Below is a notebook snapshot from developing the model. You can follow the link to use the tool. You will also need to download a fake transaction file which is also provide for you to input. The tool does take some time to load due to the hosting services processes, but it will load after a few minutes.
